## Getting Started

To get a local copy up and running follow these simple example steps:

### Prerequisites

clone repo: `https://github.com/Gegardus/JS-Capstone-Webapp`

then
`cd JS-Capstone-Webapp`

### Install

run `npm install` to install dependencies
